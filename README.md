# OSUM_VSL

## Description

OSUM VSL

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# run migration
$ yarn typeorm migration:run

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
